# Changelog

All notable changes to `cirron-sdk` are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

`0.x` is unstable — minor versions may include breaking changes until `1.0.0`.

From `0.0.1` onward, releases are produced by [`auto`](https://intuit.github.io/auto/)
on merge to `main`. This file is maintained automatically; see PR labels for the
release type.
